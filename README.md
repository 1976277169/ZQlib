# ZQlib
Updated 2018-06-05

ZQlib是左庆在北航读博士期间累积的一个库，里面包含各种小工具，在验证算法时可以很快的拼凑出初稿。这并不是一个高性能库。
此库会持续更新。
ZQlib里面全是头文件，有些不需要依赖其他库，有些需要依赖opencv，请使用者根据实际情况添加相应的第三方库。
